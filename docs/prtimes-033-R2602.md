# チャットで答える新感覚アンケート「AIQ」を採用！AICUが国内最大級の生成AI利用実態調査を実施、参加者には10,000ポイント付与

> わずか3ヶ月でAIなしでは仕事ができないクリエイターが9割に。AIチャット型アンケート[AIQ]で爆速AIDX体験。参加者だけが見られる結果速報ページも公開。

- **発表日**: 2026年2月19日 14時58分
- **発表元**: AICU Japan株式会社

AICU Japan株式会社（本社：東京都港区、代表取締役：白井暁彦、以下「AICU」）は、「生成AIクリエイターの創作プロセス・権利意識・制度ニーズを定量的・統計的に把握」する大規模調査を2026年2月19日より開始しました。一般財団法人デジタルコンテンツ協会（DCAJ）との協力により、生成AIの安全性確保に向けたリスク探索・低減および学術研究・政策提言に活用することを目的としています。

**調査URL: https://p.aicu.jp/R2602**

## 生成AIクリエイターの「声なき声」を可視化する

AICUはこれまで、書籍出版、クリエイターコミュニティ醸成、AIDX教育・コンサルティング、および国際的なパートナーシップを通じて生成AI普及に貢献してきました。しかし「実際に生成AIを使って創作している当事者(AIクリエイター)たちが何を感じ、何を必要としているのか」という定量的なデータが国内ではほぼ存在しないという課題があります。

著作権問題、無断学習、収益構造の変化、AI生成物の作者性——これらの議論は加速していますが、当事者であるクリエイターの定量的な声なしに、適切な政策やルールは設計できません。

## 前回調査（R2511）の主要データ

2025年11月実施の第1回調査（n=53）から：

- 「AIなしでは仕事が成り立たない」が53%、91%がAI不可欠と認識
- 女性比率42%（業界平均を大きく上回る）
- 「時間短縮」達成は90%、「新規受注」獲得は期待段階

## 4つの調査テーマ

### 1. 創作プロセスとAIの関係
生成AIがクリエイターのワークフローにおける役割を構造的に把握。

### 2. オリジナリティと作者性の再定義
プロンプト工夫、選別、仕上げなどからAI時代の作者性の根拠を定量化。

### 3. 権利・報酬・制度への要望
無断学習への態度や収益構造変化、必要な法制度について調査。

### 4. 人間のクリエイターにしかできないこと
意図、責任、文脈、身体性など13の選択肢からクリエイターの自己認識分布を明らかに。

## AIQ（AIチャットアンケートシステム）について

LINEチャットボット「全力肯定彼氏くんLuC4」での3年以上の運用経験から開発。従来フォーム形式ではなく一問一答の対話形式で、「超高速な応答速度(当社規定:5秒以下)」を実現。所要時間は約3分。
